# 🌦 Weather App

A dynamic weather application that fetches and displays real-time weather data using JavaScript and REST APIs.

## Features

* Search weather by city
* Get weather using location
* Shows temperature, humidity, and wind
* Temperature chart
* Dark mode

## Setup

1. Get API key from OpenWeather
2. Replace in code:

```
const apiKey = "YOUR_API_KEY";
```

3. Open `index.html` in browser

## Tech Used

* HTML
* CSS
* JavaScript
* REST API
* Chart.js
